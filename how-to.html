GraphQL

How to use GraphQL

GraphQL is a query language for an API. Follow the link for more information: https://graphql.org/

Authentication

Endpoint:

POST /graphql

Headers:

Authorization: Bearer <access token>

Main API methods that you need

  1. Get all Personas(bots)

    Query personas

    It will also return chatId for personas with whom the user has a created chat. Otherwise, field will be null.

  2. Get user information

    Query user

  3. Get all chats and listen for new message updates(Websocket)

    Subscription chatList

  4. Get chat messages

    Query chatMessages

  5. Send a message to the chat

    Mutation sendMessage

  6. Create a chat with a personas

    Mutaion createChat

    Every user can create only one chat with each persona.

For the full information about the parameters and response see the link below.
