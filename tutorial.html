Intimate.ai Authentication

In this section we will show how to authenticate users and get access tokens.

Before you start

Make sure that you have:

  • Base URL: URL to the server

  • Client API token: Token is given by Intimate.ai to access our API

How Authentication works

  1. Every user needs an access token to access the API

  2. Access token can be obtained using client API token

  3. Every access token is associated with specific user and lives forever

  4. To obtain an access token you have to provide unique user id, that identifies a single user in your system.

Getting the Access token

Endpoint:

POST /b2b/token

Headers:

Authorization: Bearer <Client API token>

Request body: Content-type: application/json

{ "provider_id": "unique_user_id" }

Response:

{ "token": "access_token_here", "is_new_user": true }

In the response you can also check if it was newly registered user or not.
